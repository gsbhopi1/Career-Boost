# CareerBoost – Student Opportunity Platform

CareerBoost is a web platform designed to help students discover scholarships, internships, and college recommendations in one place. The project focuses on providing a responsive and user-friendly experience with filtering, authentication, and backend integration.

## Features

* Scholarship listing and filtering
* Internship search and sorting
* College recommendation system
* Firebase login and signup authentication
* AI chatbot powered by Gemeni API
* Responsive user interface
* Backend API integration using Node.js and Express.js
* MySQL database connectivity

## Tech Stack

## How to Run

### 1. Clone the Repository

git clone https://github.com/gsbhopi1/Career-Boost.git

### 2. Install Dependencies

npm install

### 3. Setup MySQL Database

Create a MySQL database and configure the credentials inside:
server.js

### 4. Run the Server

node server.js

### 5. Open the Project

Open:
index.html

or run through localhost.

## Note

The backend uses a local MySQL database, so the complete project is not currently deployed online.
